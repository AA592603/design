## Usage

Skeleton avatars may be used to improve perceived performance by rendering a rough representation of an avatar in a block of content that is still loading.

## Options

Skeleton avatars support the same variants as the [avatar](/components/avatar) component.

## Acccessibility

Refer to the loading pattern [accessibility guidelines](/ui-patterns/loading#accessibility). Specifically, the secion about [avoiding over-announcing loading states](/ui-patterns/loading#avoiding-over-announcing-loading-states).
